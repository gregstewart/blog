--- layout: post title: Default CSS tags: [] status: publish type: post published: true meta: {} --- Hat tip to the webreference news letter which pointed me to an article over at builder.com, which looks at default stylesheets. I did not realise that most browsers have an in-built default style sheet that helps control the default behaviour of html pages (such as hiding the head tag or by default all tags should be displayed as block elements [which if removed amusingly displays everything in-line]). It goes on to point to several sources (Eric Meyer, Tantek and Andrew Krespanis) who have looked more closely at the html.css default browser stylesheet (this is the default stylesheet for FireFox I believe) and carried various little experiments.
