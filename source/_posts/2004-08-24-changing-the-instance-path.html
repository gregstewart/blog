--- layout: post title: Changing the instance path tags: [] status: publish type: post published: true meta: {} --- Another item to add to the list of JRun/cfmx configuration tips. If you wish to change the default path for your CFMX instance, then you need to edit the jrun-web.xml in cfusion_ear\cfusion_war\WEB-INF folder. Look for the <virtual-mapping> and add something like the following that will reflect your instance's new root path. [code] / c:/mynewrootdir [/code] Thanks Simon for sharing this tip!
