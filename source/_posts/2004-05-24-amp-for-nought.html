After making the header change I decided to re-validate my pages, and shock horror they were valid. No fault of the code embedding the Flash, but rather my sloppy blogging technique.

I built the CMS that drives this site 2 years ago and built it purely with flexibility in mind. Truth be told I think it served it’s purpose well in this capacity. Times have moved on and faced with content that doesn’t validate (even if the template framework is valid), I have decided to re-build &amp; (that’s the CMS).

So as my first step in re-vamping the CMS I thought I’d put out version 1 for nought. Yep you can download it, install it, use it, do whatever you like with version 1. I am hoping that some of you may find it useful and maybe you can provide me with some feedback or even suggestions as to where version 2 should head.

For a full list of what 1.0 can do, have a look at this page

For version 2 here are a few things that I intend to add/enhance:

  • generate/store xHTML valid content (and deal with all those special characters)
  • add a spell checker (probably Jazzy)
  • preview content
  • improved content management
  • revamp the entire interface using CSS and xHTML (a first taster is the nice logo designed by Martin Cleave)
  • build it using ColdFusion (New Atlanta’s version, though it should be fully compatible with CFMX)
  • base it on the Mach-ii Framework
  • better install scripts and documentation

I am sure the lost will grow, but these so far have been the goals I have set myself for this project. In the meantime, hope you like version 1. Grab the source here! Do note that it doesn’t contain much in the way of documentation and only possess rudimentary install scripts. Drop me a line if you need any help.
