This weeks mail bag contained a few gems

  1. Jakob Nielsen's review, the pontif of all things usable, of a book on persuasive design. Check it out here: http://www.useit.com/alertbox/20030303.html and remember it's for the kids!!!
  2. Over at the register a snippet about ten common e-commerce site flaws. I don't think one can ever have enough check lists to check our work against! Slightly related is this list of top 20 vulnerabilities and since we all need web servers there is some info on specific vulnearbilities for IIS and Apache
  3. Starting a business? Thinking of becoming an entrepreneur?? This interview with Constance Bagley @ the Harvard Business School web site has some useful information
  4. And now for stuff I came across on Zeldman's site: Check out this link to http://www.hiveware.com/enkoder_form.php. A script that takes your "mailto:" address and turns it into encrypted string and uses Javascript to write it to the page. Pretty marvellous and that should hinder some of those nasty little spam bots gleaning mail addresses from web pages. A nifty and low effort alternative to using forms to mail enquiries to you!
  5. http://www.easyrgb.com/harmonies.php, if you are like me and struggle with colour schemes for web sites, try this script. Absolutely fantastic!
  6. a little bit of light entertainment: check out fly guy. Lovely animation and a very soothing experience
