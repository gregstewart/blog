--- layout: post title: Web forms tags: [] status: publish type: post published: true meta: {} --- Dan Cederholm has published another great overview of how to build accessible forms. He takes a form built using the "traditional" method, i.e. tables, and during the course of the article shows you how to re-create it using guidelines set out by web standards (interestingly he opted for definition lists, whereas I have only ever thought of using divs and spans). To cap it off he applies some neat little css to enhance the look and feel.
