Below are three useful load testing tools that were mentioned in a Mach-ii thread. I was only aware of one so far: MS Web Application Stress Tool. Today I was made aware of two other options:

  1. Apache JMeter, a java based desktop application that can not only load test your web applications, but also databases, ftp servers and java objects.
  2. OpenSTA an open source load testing solution, which runs on Win32 clients.
