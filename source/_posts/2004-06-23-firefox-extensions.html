Via Sean Corfield over to Dan Switzer and thanks to both of you for a great list of FireFox extensions.

  1. Tabbrowser Extensions
  2. Having played around with Aggreg8 as an RSS aggregator and not being 100% satisfied I tried this one: RSS Reader Panel
  3. Ever felt the need to check something out in IE? Here's an extension for you then: ieview
  4. This one is superb as well: EditCSS, which allows you to edit currently loaded style sheets
  5. SpiderZilla, off line browsing alternative for FireFox
  6. From the guys behind the web developer tool bar. another useful extension: User Agent Switcher
