Just as I had settled for a background I visited to Cameron Moll’s blog and then I noticed it, I had chosen the same background as he had. Funny how the mind works, so I went back to Squidfingers and chose a new pattern and also chose a new color.

I also fixed an awstats configuration problem. I was using a shell script to build both the index page and a set of static pages and when it came to building the static pages the process would abort with the following error message:

awstats.pl /usr/bin/perl: bad interpreter: Permission denied

The solution: the awstats.pl script did not have execute permissions at a group level.

As a result I have now been able to see who's been looking at my site and was pleased to see a nice spread of browsers and OSs. Which made me wonder how the site renders. I have to be honest I basically only use FireFox and occasionally look at it in IE 6.0 (and usually spot a few errors). So I was wondering if possibly some of you who look at this site could and send me a screen grab if you spot any display errors? Ta, in advance.
